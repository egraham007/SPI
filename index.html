Methodology

How SIR is calculated

A single transparent formula built around how swimmers actually contribute at conference championships.

STEP 01
Percentile mapping

Each time is mapped against the conference's percentile distribution for that event and season.

STEP 02
SIR score

Scores run from 1.000 (best) upward — like golf. Lower is better.

STEP 03
Context shift

The same SIR process runs independently for each conference or division of swimming using the best times in the conference. A swimmer ranked #42 in the ACC might rank #3 in the Patriot League — that's the point.
